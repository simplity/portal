Contract Maintenance

A simple web page where user you can get, filter, add, update the contract details along with its related children data(bench, storage, location, roles). This example involves Micro Services implementation with Swagger OpenAPI Specification and Protobuff as the format to interchange the data between the services.

Pre-requisites

  • JDK 1.7 or later
  • Maven 3.0+
  • Apache Tomcat 1.7 or later

How to run this application

This application has 3 modules. Each module is using different environment to start the application.Following are the list of modules under this application:
  • using Springboot
  • using Grizzly
  • Web application with Tomcat
Build and run the jars using maven.Once application starts running check the functionality of each module with these URLs:
localhost:8080/index.html
to access spring boot application
localhost:8070/webapp/index.html
to access grizzly application
localhost:8080/webapp/
to access web application
Spring boot module
src/main/java/org/simplity/examples/contractDB/springBoot/App.java
is the entry class to run Springboot application
sf
src/main/java/org/simplity/examples/App.java
is the entry class to run application with grizzly server

Services

contractCreate
inserts new contract entry into DB along with its children
contractUpdate
updates the contract details
contractFind
fetches all the contract details
contractGet
gets the contract details for the provided contractID
contractFilter
filters the contracts based on the search criteria
