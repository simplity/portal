petClinic

A web-application to view,add and edit the details of veterinarians, pet owners,pets etc.

Pre-requisites

  • JDK 1.7 or later
  • Maven 3.0+
  • Apache Tomcat 1.7 or later

How to run this application

  • Download and unzip the repository for this example or clone it using Git:
    git clone https://github.com/simplity/examples.git
  • cd into petclinic
  • build the war file using maven:
    mvn clean install
    installs and run the scripts of embedded DB and generates the war file in the target folder of the application
  • deploy the generated war(petstore.war) in Tomcat server and start the server.
Wait for the server to start, go to the browser and try it out:
http://localhost:8080/petstore

You can see the petstore page where you can find the list of veterinarians and also search for a particular pet owner and edit it.
FindOwners tab allow you to search for owner/owners and also can add a new owner details

Services

filterOwners
filters all the owners along with the petDetails for each owner based on the input search text
filterOwnersJava
It invokes java class which performs DB operations to filter owners

On-the-fly Services

Simplity on-the-fly services can also be used to achieve the same functionality as above without writing any service.
filter_owners
filters all the owners based on the input search text
get_owners
list of all the owners
save_owners
add/updates the owner details based on the existence of primary key in the corresponding table
filter_vets
filters/list of all the veterinarians based on the input search text
save_visits
add/updates the pet visit details
get_petDetails
list of all the pet details
save_pets
add/updates the pet details
list_petTypes
list of all the pet types
