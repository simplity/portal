Spring integration with Simplity

A simple application that fetches data and display it as a table.This example gives an overview of integrating Spring Framework with Simplity.

Pre-requisites

  • JDK 1.7 or later
  • Maven 3.0+
  • Apache Tomcat 7.0+

How to run this application

  • Find the source code for this project at
    https://github.com/simplity/examples/tree/master/springIntegration
Build the war file using maven and deploy it in Tomcat server.Once the server start and service is up visit http://localhost:8080/springIntegration/
Click on the 'GetUsers' button to view tha data.

Configurations

/springIntegration/src/main/java/org/simplity/examples/springIntegration/logic/FetchUsers.java
is the logic class gets invoked by service fetchUserRecord to interact with Spring.
classManager is the attribute that needs to be set in
application.xml
and it's value is a java class that should implement
org.simplity.tp.ContextInterface
to inject the Spring context into the Simplity.
