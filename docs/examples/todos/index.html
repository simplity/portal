To-do List

A simple web-application which displays a list of to-do items.One can add or update the to-do item. Also can view the active and completed to-do items. This example involves JMS implementation with Active MQ as messaging server.

Pre-requisites

  • JDK 1.7 or later
  • Maven 3.0+
  • Apache ActiveMQ

How to run this application

  • Find the source code for this project at
    https://github.com/simplity/examples/tree/master/todos
This application has 4 modules. All modules should be running to start the application.Following are the list of modules under this application:
  • activeMQ - starts the Apache ActiveMQ server
  • todoService - handles the requests to add/update/delete and write the data to JMS.
  • todoViewService - handles the request to get data from DB. This service keeps listening to JMS and performs the CRUD operations on DB with the consumed data.
  • todosUI - provides UI to perform all the actions.
Build and run the jars using maven.Once the services are up and running visit http://localhost:8083/index.html where you'll see the list of all the to-do items.
All the configurations related to DB and JMS are done in application.xml file.

Services

addTodo, updateTodo, deleteTodo
writes the to-do details along with the corresponding action to the configured JMS queue.
viewTodos
fetches all the to-do items from the DB
jmsConsumer
consumes the data from the configured JMS queue and redirects the data to "crudAction" service.
crudAction
performs all the DB transactions like add, delete & update
