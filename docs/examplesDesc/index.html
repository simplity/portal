Simplity sample example

This application is a simple java application which gives an idea of the folder structure of simplity components and the servlet entry to invoke the simplity features. In this example you can see how to add service invoking the same.

How to run this application

  • Download and unzip the repository for this example or clone it using Git:
    git clone https://github.com/simplity/examples.git
  • cd into mcHelloWorld
  • mvn clean install
  • cd into target
  • java -jar mcHelloWorld-1.2.1-SNAPSHOT.jar
Wait for the server to start and go to the browser and try it out http://localhost:8081/a._s?_serviceName=helloworld
You will see the output on the browser window.
Try writing your own services in comp/service/tp folder of the application and call it using the above url with corresponding service name.
