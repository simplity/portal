mcHelloWorld

This example is a simple java application which default run on '8081' port which you can configure in the application's Main class. org.simplity.http.Serve is the default servlet of simplity to deliver the services. In this example you can see how to add service and send back the response to the client.

How to run this application

  • Download and unzip the repository for this example or clone it using Git:
    git clone https://github.com/simplity/examples.git
  • cd into mcHelloWorld
  • mvn clean install
  • java -jar target/mcHelloWorld-1.2.1-SNAPSHOT.jar
Wait for the server to start, go to the browser and try it out:
http://localhost:8081/helloworld?_serviceName=helloworld

_serviceName
is the query param to invoke the corresponding service.
In response to this request, you will see a JSON response sent by service "helloworld", representing a greeting, that looks something like this:
{"hello":"Hello World! from simplity","_messages":[]}
Try writing your own services in comp/service/tp folder of the application and call it using the above url with corresponding service name.
