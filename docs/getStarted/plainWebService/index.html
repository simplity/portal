Plain Web Services
Simplity offers in-built web services. You can directly call the Simplity services by using this web service.
Steps to create Plain Web Service using Simplity
1. Get simplity kernel
For Maven users-
  • Copy code
<dependency>
<groupId>org.simplity<groupId>
<artifactIdId>simplity</artifactId>
<version>1.2</version>
</dependency>
Plain Java-
  • Copy code
<dependency>
<groupId>org.simplity</groupId>
<artifactIdId>simplity</artifactId>
<version>1.2</version>
</dependency>
2. Creating project
  • Create a new "Maven Project" in eclipse
  • Configure web.xml to have the following entry
    • Copy code
    <servlet>
      <servlet-name>SimplityStartup</servlet-name>
      <servlet-class>org.simplity.http.Startup</servlet-class>
      <load-on-startup>1</load-on-startup>
    </servlet>
    <servlet>
      <servlet-name>s</servlet-name>
      <servlet-class>org.simplity.http.Serve</servlet-class>
    </servlet>
    <servlet-mapping>
      <servlet-name>s</servlet-name>
      <url-pattern>*._s</url-pattern>
    </servlet-mapping>

3. Application Configuration
  • Create component folder structure under WEB-INF as follows:
    • api
    • comp
      • dt
      • msg
      • rec
      • service
        • tp
      • sql

  • Configure application.xml under WEB-INF/comp as follows.
    • Copy code
    <?xml version="1.0" encoding="UTF-8"?>
    <application
            xmlns="http://www.simplity.org/schema"
            applicationId="HelloWorld"
            autoLoginUserId="100"
            logSqls="true">
    </application>
      
    • This is the root file of simplity which helps to configure various features of Simplity. Here "applicationId" is a mandatory field which represents the application name. applicationId should be a unique name for the application.
    • NOTE: To get more help, click ctrl + tab from the eclipse editor.
  • Create Simplity service under WEB-INF/comp/service/tp as follows.
    • Copy code
    <?xml version="1.0" encoding="UTF-8" standalone="no"?>
    <service xmlns="http://www.simplity.org/schema"
            name="myWebService"
            dbAccessType="none">
        <actions>
            <setValue fieldName="hello" fieldValue="from within restService Hello World!"></setValue>
        </actions>
        <outputData fieldNames="hello"></outputData>
    </service>

3. Invoke Simplity Web Service
  • Enter the below URL from the web browser,
    • Copy code
    "http://localhost:8080/helloworld/a._s?_serviceName=myWebService"

    • Note :
      • helloworld - application name.
      • myWebService - simplity service name.
4. Simplity Web Service client
  • Simplity also provides web service client and it can be achieved using httpClient tag.
  • Create Simplity client as follows.
    • Copy code
    <?xml version="1.0" encoding="UTF-8" standalone="no"?>
    <service xmlns="http://www.simplity.org/schema"
            name="simplityWebService"
            dbAccessType="none">
        <actions>
            <httpClient httpMethod="GET" urlString="http://localhost:8080/helloworld/a._s?_serviceName=helloworld"
                    contentType="application/json" responseFieldName="hello">
            </httpClient>
            <log names="hello" />
            <setValue fieldName="hello" expression="concat('from simplityWebService: ',hello)"></setValue>
        </actions>
        <outputData fieldNames="hello"></outputData>
    </service>

  • Invoke Simplity service using javascript as follows.
    • Copy code
    callWebservice() {
        Simplity.getResponse("simplityWebService", null, onSuccess, onError);
    }

    function onSuccess(response) {
        //display the response;
    }

    function onError(response) {
        //display the error message;
    }
