Simplity Service with Java program
Invoke Simplity service from Java program.
Steps to call Simplity service from Java program
1. Get simplity kernel
For Maven users-
<dependency>
<groupId>org.simplity<groupId>
<artifactIdId>simplity</artifactId>
<version>1.2</version>
</dependency>

Plain Java-
<dependency>
<groupId>org.simplity</groupId>
<artifactIdId>simplity</artifactId>
<version>1.2</version>
</dependency>
2. Creating project
  • Create a new "Maven Project" in eclipse
3. Application Configuration
  • Create component folder structure under WEB-INF as follows:
    • api
    • comp
      • dt
      • msg
      • rec
      • service
        • tp
      • sql


  • Configure application.xml under WEB-INF/comp as follows.
    <?xml version="1.0" encoding="UTF-8"?>
    <application
            xmlns="http://www.simplity.org/schema"
            applicationId="ContractService"
            autoLoginUserId="100"
            logSqls="true">
    </application>
      
    • This is the root file of simplity which helps to configure various features of Simplity. Here "applicationId" is a mandatory field which represents the application name. applicationId should be a unique name for the application.
    • NOTE: To get more help, click ctrl + tab from the eclipse editor.

  • Create Simplity service under WEB-INF/comp/service/tp as follows.
    <?xml version="1.0" encoding="UTF-8" standalone="no"?>
    <service xmlns="http://www.simplity.org/schema"
            name="myService"
            dbAccessType="none">
        <actions>
            <setValue fieldName="hello" fieldValue="Invoking Simplity service from Java program - Success"></setValue>
        </actions>
        <outputData fieldNames="hello"></outputData>
    </service>

3. Write Java program to invoke Simplity Service
  • Simplity provides JavaAgent to call simplity service from the java program.

    import org.simplity.kernel.Application;
    import org.simplity.service.JavaAgent;
    import org.simplity.service.ServiceData;

    public class SimplityServiceWithJava {

        public static void main(String[] args) throws Exception {
            String comp = "D:/GITHub/examples/helloworld/src/main/resources/comp";// Simplity comp folder path
            Application.bootStrap(comp);
            String serviceName = "myService";
            JavaAgent agent = JavaAgent.getAgent("100", "");
            ServiceData outData = agent.serve(serviceName, "");
            System.out.println(outData.getPayLoad());
        }

    }
